# 🍽️ Restro POS

A modern and simple full-stack POS system for restaurants and cafés — built with **React**, **Tailwind**, **Express**, **MongoDB**, and **Stripe**.

---

## 🚀 Features

- 🧑‍🍳 User authentication & role-based access  
- 🪑 Table & order management  
- 🍔 Menu management  
- 💳 Stripe payments integration  
- 🌐 REST API with Express  
- 💾 Data stored in MongoDB  
- ✨ Clean UI with React + Tailwind  
- 🔐 JWT & cookies for secure auth  
- ⚡ Optimized with React Query & Redux Toolkit  

---

## 🛠️ Tech Stack

**Frontend:**  

- React 19, React Router 7  
- Redux Toolkit  
- Tailwind CSS  
- Stripe.js  
- Vite  

**Backend:**  

- Node.js + Express  
- MongoDB + Mongoose  
- Stripe API  
- JWT, bcrypt  
- dotenv, CORS  

---

## 📦 How to Run

**Backend**  

```bash
cd backend
npm install
npm run dev
```

**Frontend**  

```bash
cd frontend
npm install
npm run dev
```
